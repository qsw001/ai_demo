I will implement an `AudioSystem` class using the Web Audio API to provide sound effects and background music without external assets.

## 1. Create `js/AudioSystem.js`
This module will handle all audio generation.
*   **Context**: Create `AudioContext` lazily (on first user interaction).
*   **BGM**: Use an oscillator-based sequencer to play a simple looping melody (bassline).
    *   Notes: C2, G2, A#2, F2 sequence.
    *   Waveform: Triangle/Sawtooth with low pass filter.
*   **SFX**:
    *   `playEat()`: Sine wave slide (600Hz -> 1200Hz, 0.1s).
    *   `playDie()`: Noise burst or Sawtooth slide down (200Hz -> 50Hz, 0.5s) with gain decay.
    *   `playEnemySpawn()`: Low frequency oscillator slide up (100Hz -> 300Hz) or a "warning" pulse.
    *   `playEnemyDie()`: Crunchier noise burst.

## 2. Integrate into `js/Game.js`
*   Import `AudioSystem`.
*   Instantiate `this.audio = new AudioSystem();` in constructor.
*   **Initialize Audio**: In `handleSpace()` (first user interaction), call `this.audio.init()` and `this.audio.startBGM()`.
*   **Triggers**:
    *   In `checkCollisions`:
        *   Eat food -> `this.audio.playEat()`
        *   Kill enemy -> `this.audio.playEnemyDie()`
        *   Player die -> `this.audio.playDie()`
    *   In `updateEnemies`:
        *   Spawn enemy -> `this.audio.playEnemySpawn()`
    *   In `gameOver`:
        *   Stop BGM -> `this.audio.stopBGM()`

## 3. Implementation Details
*   **Web Audio API**: Use `OscillatorNode`, `GainNode`, `AudioContext`.
*   **Safety**: Handle browser autoplay policy (resume context on click/key).
